# AGENTS (backtap_analyzer)

Scope: /Users/yusuke/Documents/GitHub/backtap_analyzer

## Commit Journal Rule

- Before every commit, append an entry to `/Users/yusuke/Documents/GitHub/backtap_analyzer/docs/user-comment-journal.md`.
- Each entry should include:
  - date and time with timezone (ISO 8601)
  - what the user requested/changed
  - what was implemented or decided
- Keep entries short and chronological.
- If the user explicitly asks not to log a commit, follow the user request.
- Recommended heading format:
  - `## Entry: YYYY-MM-DDTHH:MM:SS+09:00 (JST)`

## Purpose

This rule keeps a continuous memo stream for later publishing (for example, Zenn articles).
